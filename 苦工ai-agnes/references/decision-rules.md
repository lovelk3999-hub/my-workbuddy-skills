# 苦工AI决策规则

## 核心原则

**不要用主AI的token干体力活。**

## 决策流程

```
当前任务
  ↓
需要写代码/改模板/调浏览器/操作文件/Git？
  ├→ YES → 主AI自己干（需要推理和环境操作）
  └→ NO  ↓
需要复杂判断/选方案/设计架构？
  ├→ YES → 主AI自己决策（需要全局context）
  └→ NO  ↓
     → 叫苦工Agnes！🎯
       ├→ 文本任务 → agnesChat / agnesBatch
       ├→ 图片任务 → agnesImage / agnesImageEdit
       └→ 视频任务 → agnesVideo / agnesVideoFromImage
```

## 典型场景

### 文本体力活 → 叫苦工

| 交给苦工（Agnes文本） | 主AI自己干 |
|:---------------------|:-----------|
| 数据格式化成JSON | 写generate.py代码 |
| 批量写FAQ/描述 | 改HTML模板 |
| 翻译内容 | 控制Chrome搜网页 |
| 总结/提炼要点 | Git操作/部署 |
| 整理定价/功能数据 | 决策：选哪个工具 |
| 生成meta/OG标签 | 架构设计 |

### 图片体力活 → 叫苦工

| 苦工能做的图 | 说明 |
|:-------------|:-----|
| 文生图 | 根据文字描述生成配图、封面、插图 |
| 图生图-风格迁移 | 把照片变成插画/赛博朋克/水墨风 |
| 图生图-场景转换 | 白天变黑夜，夏天变冬天 |
| 高密度信息图 | 复杂场景/密集元素/城市景观 |
| 产品图美化 | 白底图、场景图、氛围图 |

### 视频体力活 → 叫苦工

| 苦工能做的视频 | 说明 |
|:--------------|:-----|
| 文生视频 | 根据文案直接生成短视频素材 |
| 图生视频 | 让静态图片动起来（微动效） |
| 多图视频 | 多张图片间平滑过渡 |
| 关键帧动画 | 关键帧之间生成流畅动画 |

## 审核三部曲

1. **L1 自动校验**：validateJson()检查JSON格式+字段完整；图片视频检查URL是否可访问
2. **L2 主AI抽检**：随机抽20%看质量，有问题反馈重做
3. **L3 人类标记**：上线前标记[NEEDS_REVIEW]

审核失败：最多重试3次，仍失败则[NEEDS_USER]暂停。
